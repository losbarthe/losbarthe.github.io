Choropleth Map

Below you can view an interactive map of SDG indicator 11.1.1 (on the left), which I created using publicly available SDG data found through the UN database. This can be compared to SDG indicator 3.1.2 (on the right) found in the ESRI-online database. Indicator 11.1.1 depicts the proportion of urban populations living in slums, while indicator 3.1.2 shows the percentage of women of childbearing age within 2 hours of a health facility. When clicking on specific regions, a popup depicts the value of the indicator for that specific region.

A choropleth map is a thematic map that uses differences in coloring/shading within predefined areas to represent the average values of a particular quantity in those areas. It visually communicates data patterns and trends over geographic regions, making it easier to compare different areas at a glance. Choropleth maps are valuable tools for spatial analysis and decision-making. In designing choropleth maps, the color scheme should be chosen to best represent the information presented and convey the intended message and the amount of classes should not exceed 5 so as to keep the information digestable. This style of map is useful as it is rather common and therefore more easily and uniformly interpreted.

Another consideration is the projection used in mapping. Although coropleth maps are very useful, the prominence of areas is largely correlated to the area these regions take up on the map. Below you can find an image of an equal area projection map of the same data represented above.

While this equal area projection is useful in depicting all regions equally in area this type of map is not able to adequately show data for smaller regions. In that case, consider a graduated symbol map, which gives each area a different sized symbol (based on the value of that indicator in that region) no matter the size of the region itself.
